# 2. Installation Guide

**System Requirements:**
- OS: Windows 10+, macOS 11+, Linux
- RAM: 2 GB minimum
- Disk Space: 50 MB

**Installation (Windows):**
1. Download installer
2. Run the file
3. Follow setup wizard
4. Finish

**Installation (macOS):**
1. Download `.dmg` file
2. Drag to Applications
3. Launch app
